Intro to Javascript:

The programming language used to bring HTML and CSS to life.

19/09/2020

Last week for EDA foundations we spent time learning HTML and CSS.
(Hypertext Markup Language and Cascading Style Sheets)

I began to develop my understanding of these languages and what we can achieve with them.
This blog is made with HTML and CSS (and with the help of a CSS framework 'Bootstrap'). HTML is made up of various tags that follow a structure. We can enclose information like text and images in these tags, and our browser will interpret these tags and display our page.

HTML alone looks incredibly plain, basically like a simple Word document. We can style our HTML with CSS. Our CSS document will target "IDs" and "Classes" that we defined in our HTML. Once we have targeted these elements, we can begin to modify them and apply styling. We can style anything from background colour, font style, size and colour, we can position elements, and modify their margin, border, and padding. This just scratches the surface of possibility!

I began to imagine these concepts and how they might relate to something like a house. HTML could be the structure and foundation of the house, and CSS could style the wallpaper, rooms, and positioning of furniture around the house. Now you go into a room and reach for the light switch. Let's introduce Javascript!

Javascript

Javascript (JS) is a programming language that is used in web development. We can define variables, store data in the form of objects and arrays, create and execute functions, and ultimately interact with our HTML and CSS.

Control Flow

Control flow is a foundational principle in programming including JS. It is the order that the code we write is executed. Our code can include conditional statements, loops, and functions. When the control flow reaches a conditional statement, an example in JS would be an 'if/else conditional statement', the flow will run through it, and it will function in accordance to how we created it.
If we go back to the house example, a simple way I can describe an if/else statement could be in the following way:

  1. Enter a room,
  2. If you need light, turn on the light.
  3. Otherwise (else), open the curtain.

Loops

Loops are used to run code until the conditions we specify are met. A 'for loop' first initializes the loop, it then checks the condition it needs to meet so it can stop looping, and then we define how we update the loop after each iteration. Let's say we need 4 walls to build our home. We could run a for loop that has if/else statements enclosed that updates and reports back each time a wall was built, until we have 4 walls.

  1. We start with 0 walls. We need 4.
  2. Announce the wall building stages until we have 4 walls.

Functions

We can enclose the for loop from the previous example in a function. Functions are useful in javascript, they are blocks of code that perform a specified task. We can "call" them to perform their task when we want them to run. We can create paramaters for the function, and then call it back with arguments. When the previous example is created into a function, it will perform the task when the control flow reaches the stage where the function is called. The control flow will jump back to where the function was defined and execute from there. We can also call manually in the console. Arrays and Objects

An array enables us to store a collection of data entries in a single variable. We can add and remove from an array [ ], as well as read the contents. The first element of an array is indexed at 0. The image below demonstrates how we can access the data in an array.

Objects in JS can be thought of similarly to how we describe objects in real life. We define a variable and make it an object { } that contains properties and their description. We refer to the properties as "keys", and the description as "values" (key value pairs). These are essentially variables in an object. Similar to arrays, we can access the data, as well as manipulate it.

The DOM

The Document Object Model (DOM) is what ties our HTML and CSS with Javascript. It is an interface for our web pages, known as an application programming interface (API). It is what enables our browser to interpret and render our code. We can access the DOM and manipulate it with JS, allowing us to change our HTML and CSS on demand. That means we can modify our content, structure, and styling of our web page.
