The CSS Box Model:

Margin, border, and padding.

11/09/2020

Understanding the CSS box model will enable us to effectively organise the layout and design of our websites. I will be demonstrating this concept with regards to 'block level' HTML elements (elements such as: paragraphs, headers, forms, tables, divs) as it is these elements the CSS box model is more commonly applied to.
Imagining packing a box in a room helped me grasp this concept! First I will start with some content that will be inside the box.

Content

Our content is simply text inside a div with the class "box". This could also be any other block level HTML element. I have set the width and background colour for this div to help with the demonstration. Now we can begin to define the contents surroundings.

Border

I like to imagine that the border is the cardboard we are packing the content into. We can change the thickness, colour, and style of the border (cardboard) in CSS using the following border property shorthand:
border: border-width border-style border-color

Padding

We can add padding around our content and this will space the content away from the border. Like adding bubblewrap to an item before it goes into a box! You will notice that adding the padding (and border) has added dimension to the div, that is to do with the box-sizing property. Read up on that at W3Schools.
padding: length

Margin

Margin is the space around the border. I like to imagine that adding margin will move the box around the room, allowing us to walk around the box or reposition it. Notice how the box is pushed away from the side of the browser window.
margin: length

More info

Did you notice that before we applied our own margin to the box that there was already space between the box and the browser window? That is because our browsers automatically adds some CSS properties. If you wanted to erase that space and have the box against the browser window you would need to change the margin on the 'body' element of your CSS file and remove the margin on your box.

I would also like to mention that throughout this blog I was using the property shorthand and affected all sides with a single value. These properties have a lot of flexibility say if you wanted to isolate a particular side of the property. For example, margin-top, margin-right, margin-bottom, margin-left. The same can be applied to padding, and border. Furthermore, using the shorthand you can specify the sides you want to change on a single line. For example: margin: 10px 15px 20px 25px will add margin in the order of top, right, bottom, left.

I hope you get a sense for how the box model can be used to add borders around elements, and define space. Used alongside other foundational CSS concepts like positioning, we can begin to create effective layouts for our websites.
